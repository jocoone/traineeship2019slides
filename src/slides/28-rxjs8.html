ReplaySubject

Same as a BehaviourSubject but will remember a number of items that will be send to the next subscriber.

import { ReplaySubject } from 'rxjs';
const subject = new ReplaySubject(2); // buffer 2 values for new subscribers
subject.subscribe({
  next: (v) => console.log(`A: ${v}`)
});
subject.next(1);
subject.next(2);
subject.next(3);
subject.subscribe({
  next: (v) => console.log(`B: ${v}`)
});
subject.next(4);// Output?
A: 1, A: 2, A: 3, B: 2, B: 3, A: 4, B: 4
