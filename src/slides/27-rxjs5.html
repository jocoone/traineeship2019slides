Subject

Special kind of subject acting like both the observable and the observer

import { Subject } from 'rxjs';
const subject = new Subject();               
subject.subscribe({
  next: (v) => console.log(`A: ${v}`)
});
subject.subscribe({
  next: (v) => console.log(`B: ${v}`)
});        
subject.next(1);
subject.next(2);
// Output?
A: 1, B: 1, A: 2, B: 2
