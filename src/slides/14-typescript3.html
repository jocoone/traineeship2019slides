• Statically typed

    You can decide which type a variable should have when writing the code. From that moment the variale will/has to be of the same type. 

    let PI: number;
PI = 3.1415; // OK
PI = '3.1415'; // COMPILATION ERROR

  • Optionally

    You are not obliged to use a type for a variable. You can always use any or no type at all. 

    let PI: any;
let PI;
